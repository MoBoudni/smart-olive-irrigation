# Lastenheft für das Oliven-Bewässerungssystem

## 1. Einführung
- Projektname: Intelligentes Oliven-Bewässerungssystem
- Ziel: Automatisierte Bewässerung von Olivenbäumen basierend auf Bodenfeuchtigkeit und Wetterdaten.

## 2. Projektziele
- Optimierung des Wasserverbrauchs
- Steigerung des Ertrags durch präzise Bewässerung
- Benutzerfreundliche Steuerung und Überwachung

## 3. Funktionale Anforderungen
- Automatische Bewässerung basierend auf Bodenfeuchte und Wetterdaten
- Unterstützung mehrerer Bewässerungszonen
- Integration von Wettervorhersagen

## 4. Nicht-funktionale Anforderungen
- Zuverlässigkeit: System muss 24/7 verfügbar sein
- Benutzerfreundlichkeit: Einfache Bedienung und Konfiguration
- Skalierbarkeit: Möglichkeit zur Erweiterung um zusätzliche Zonen
