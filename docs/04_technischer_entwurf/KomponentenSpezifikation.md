# Komponentenspezifikation

## 1. Frontend-Komponenten
- **MainApp:** Hauptklasse für die JavaFX-Anwendung
- **DashboardController:** Steuert die Anzeige des Dashboards
- **ConfigController:** Steuert die Konfiguration der Bewässerung

## 2. Backend-Komponenten
- **IrrigationService:** Enthält die Geschäftslogik für die Bewässerung
- **ZoneRepository:** Verwaltet den Zugriff auf die Zonen-Daten

## 3. Datenbank-Komponenten
- **Zones:** Tabelle für Bewässerungszonen
- **Sensors:** Tabelle für Sensorwerte
